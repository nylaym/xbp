Git

ファイルの変更を管理する分散型管理プログラム

なぜ必要なのか 目的
  • 変更の記録
  • チーム開発
  • ブランチ運用
  • リモート共有

仕組み

ブランチについて


ファイルを分岐して保存できる 複数の作業を分担して行える

リポジトリについて

ファイルを保存するストレージのこと

  • ローカルgit

    • 自分のPCで使う
    • 個人開発

  • リモートgit

    • git hub,gitlabなどのサーバー

まとめ

  • gitはプログラムなどを履歴などを含め変更を管理するプログラム
    • 現在の開発には不可欠
  • ローカル、リモートなどの種類が用途ごとにある


HTML

Hyper Text Markup Language

概要

主にWEBページを作成する用途に用いられる言語
文章にタグを用いて、意味を与える

タグについて

〈h1〉見出し〈/h1〉 →のように変更を加えたい文章をタグで囲む

タグ例

  • ＜h1＞〜＜h6＞ → 見出し
  • ＜p＞ →段落
  • ＜a＞ →リンク
  • ＜img＞ →画像
  • ＜div＞ →ブロック要素の区切り

    • CSSとの違い

    CSSはHTMLを装飾するのに用いられる
    HTMLは文章の構造を決める
    CSSはデザインと見た目を決める
